Coffee Sales Dashboard Analysis

This project analyzes coffee sales data and presents insights using an interactive dashboard. The purpose of this project is to understand sales trends, product performance, and revenue patterns to help business decision-making.

Objectives:

• Analyze total sales and revenue

• Identify top-selling products

• Analyze sales trends over time

• Identify peak sales periods

• Compare category performance

Dataset Information:

The dataset includes order date, product name, category, quantity, price, and total sales.

Tools Used:

• Excel – Data Cleaning and Analysis

• Excel – Dashboard creation

• GitHub – Project hosting

Dashboard Features:

• Total Sales Overview

• Sales Trends

• Product Performance Analysis

• Category Analysis

• Interactive dashboard

Key Insights:

• Top-selling products identified

• Peak sales time identified

• Revenue trends analyzed
